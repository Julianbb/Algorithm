## 输入一整数数组，实现一个函数来调整数组中的数子顺序，使得所有奇数在前面，偶数在后面
ReOrderArray1.cpp
- 定义两个指针，　p1和p2, p1指向第一个元素，p2指向最后一个元素
- p1向后移动，碰到偶数则停下，开始移动p2,　p2碰到奇数就停下
- 交换p1和p2的数字，继续重复上步移动p1,p2操作
- 直到 p2移动到p1的前面，则完成
